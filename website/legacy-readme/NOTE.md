# Legacy README archive

`README.md` here is a verbatim snapshot of the repository's full README as of 2026-07-05
(v2.0-era, ~1,350 lines), taken when the root README was cut down to logo / what-it-is /
installation / quick start. `docs/` holds byte-identical copies of every image it references
(the root README linked them via `raw.githubusercontent.com/.../docs/...`, so the snapshot
stays intact even if the live `docs/` assets are rebuilt or renamed).

Purpose: this is the **source material for the website documentation build-out**. Every
parameter table, design note, and example in it should eventually be migrated into the site's
guides/reference pages (see `website/src/content/docs/`); the per-feature sections map roughly
one-to-one onto guide pages. Once the migration is complete and the site is deployed, this
directory can be deleted.

Do not edit the snapshot - it is a record, not a living document. New documentation goes in
the site content.
