Every chart below is **live and interactive** - hover for tooltips, use the actions menu
(top-right) to export SVG, PNG, or the Vega-Lite JSON, and follow **Open in playground** to edit the
code. All are built with dysonsphere on the standard
[vega-datasets](https://github.com/vega/vega-datasets). A dedicated molecular-biology gallery is on
the way.
