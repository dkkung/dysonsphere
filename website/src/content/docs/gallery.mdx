A tour of dysonsphere across the sciences. Every figure is **live and interactive** and built
entirely with the library. Most of the data is synthetic, chosen to show what dysonsphere can do -
the exception is **Landmark plots**, reconstructed from real open and published data. On any
figure, press **`</> code`** to read the exact code that made it, or **Open in studio** to run and
edit it live.
