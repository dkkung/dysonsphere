Every figure below is **live and interactive** and built entirely with dysonsphere - the data is
synthetic, chosen to show the library across scientific domains. Hover for tooltips; use the
actions menu (top-right of a chart) to export SVG, PNG, or Vega-Lite JSON. Press **`</> code`** on
any figure to read the exact code that made it, or **Open in studio** to run and edit it live.
