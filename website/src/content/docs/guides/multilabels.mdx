---
title: Multilabels
description: Annotation tables below the x axis - plus/minus rows, symbols, sample sizes, and grouped spans.
---

import Example from '../../../components/Example.astro';

`add_multilabel()` attaches an annotation table below a chart - `+`/`-` rows, symbols, or plain
text aligned to each x category, with optional sample-size and category-label rows and grouped
spans.

<Example name="multilabel" alt="Strip plot with a plus/minus multilabel table below" />

`showSampleSize=True` injects a per-category count row, and `categoryLabel=True` renders the
category names as an angled row of the table:

<Example name="multilabel_sample_size" alt="Strip plot with sample size and category-label rows" />

Use `style="symbol"` for filled/open dots, and `span=` to bracket groups of categories under a
shared label:

<Example name="multilabel_symbol" alt="Strip plot with a symbol multilabel table and grouped spans" />
