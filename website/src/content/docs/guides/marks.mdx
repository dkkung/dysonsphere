dysonsphere's composite marks return native Altair layers, so you compose them with `+` and keep
Altair's full API. The jitter, beeswarm, and quasirandom offsets they use are also available as
standalone [transforms](/guides/transforms/).

## Strip plot

`mark_strip()` draws a categorical scatter with a median tick and optional mean error bars. Points
are jittered by default.

## Violin plot

`mark_violin()` pairs an outlined kernel-density silhouette with Prism-style inner statistics -
a solid median line and dashed quartile lines, each spanning the violin's width at that value.
`inner=` picks the display: `"quartiles"` (default), `"median"` (median line only), `"box"` (an
embedded boxplot), or `None` (silhouette only); `innerColor` restyles the lines:

<Example name="violin" alt="Violin plot of horsepower by car origin" />

Remove the outline with `stroke=None`, and end the violin sharply at the observed data extremes
with `trim=True`; `bandwidth` tunes the KDE smoothness:

<Example name="violin_styled" alt="Styled violin plot with a custom palette, no outline, and trimmed tails" />

## Boxplot

Plain Altair marks inherit the theme too. `mark_boxplot()` gets grey boxes, a single-stroke median,
and rounded whisker caps out of the box; `boxplotOutliers=False` hides outlier points.

<Example name="boxplot" alt="Boxplot of miles per gallon by car origin" />

## Table

`mark_table()` renders a DataFrame as a publication-styled table - a composable Altair layer, not a
picture. Columns are formatted per-column (reusing the same scientific / power / SI notations as the
rest of the library, typeset with real superscripts); a numeric column can be shaded by value with
`cellColor`; and `strokes` picks which rules to draw (`"outer"`, `"header"`, `"rows"`, `"cols"`,
`"grid"` for the interior, or `"all"` for every rule). Numeric columns are right-aligned by default
so figures line up on the decimal; everything else is left. The source DataFrame is never mutated, so
`ds.read(what="data")` recovers it exactly - and every cell background is its own rect, so you can
recolour individual cells in Illustrator.
